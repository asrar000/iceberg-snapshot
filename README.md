# iceberg-snapshot

Sample CSV data for Spark/Iceberg is generated by:

```bash
python3 scripts/generate_sample_csvs.py
```

Output files are written to `data/sample_csvs/`.

CSV schema:

```text
record_id: int
customer_id: int
product_category: string
region: string
amount: double
is_priority: boolean
event_date: date
event_ts: timestamp
```
